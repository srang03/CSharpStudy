[1] 튜플
	튜플 : 괄호 기호의 간단한 구문을 사용하여 하나 이상의 속성을 가지는 객체를 만드는 형식이다.
	
	메서드의 매개 변수를 전달 할때 사용자 정의 클래스를 사용하면 한 번에 여러개의 값을 전달할 수 있다.
	튜플은 새로운 크래스를 만들지 않고 언어 차원에서 여러 개의 값을 전달할 수 있는 기능을 제공하여 편리함을 준다.

	하나 이상의 값을 전달 할 때, 사용자 정의 클래스를 만들어서 전달했는데
	튜플을 사용해서 (값 ,값, 값)의 형태로 전달할 수 있다.

	기존의 개발자들이 사용하기 편하도록 하기 위한 C# 7.0 버전의 새로운 기능 중 하나이다.
	파이썬의 영향을 받음
	
[3] 튜플 리터럴 (Tuple Literal)
	1) 기본 형식으로 튜플 객체의 아이템 접근이 가능하다.
	var r = (12, 34, 56) // 3개의 int 타입의 데이터가 r 변수에 담긴다.
	r.item1
	r.item2
	r.item3
	
	튜플의 개수만큼 item 속성이 생성된다.

	2) 사용자가 원하는 이름을 사용해서 속성을 호출한다.
	사용자가 원하는 이름을 지정해서 속성을 호출할 수 있다.
	var tuple = (x: 123, y: 456);

	3) 사용자가 원하는 이름과 형식을 지정해서 튜플 객체를 생성할 수 있다.
	(ushort x, ushort y) tuple = (123, 456);
