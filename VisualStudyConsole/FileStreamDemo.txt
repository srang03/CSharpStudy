[1] 스트림 (Stream)
	스트림 : 데이터를 파일/네트워크/데이터베이스 등으로 이동하는 메커니즘으로 영구적인 데이터 저장소에 읽고 쓰기가 가능하다.
	스트림 프로세스 : 원본 파일 > 응용 프로그램 > 저장소 (파일/네트워크/인-메모리/데이터베이스)

	Stream 또는 파일 IO를 사용해서 데이터를 저장하고 읽어오는 기능을 구현할 수 있다.
	
	파일과 디렉토리에 관련된 모든 프로그래밍 명령어는 System.IO 네임스페이스에서 사용할 수 있다.

[2] System.Data 네임스페이스
	System.Data.SqlClient : SQL Server 관련된 주요 클래스 제공한다
	System.Data : 데이터베이스 처리 관련 클래스 및 네임스페이스 제공한다
